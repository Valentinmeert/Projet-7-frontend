### Utilisation de :

- Node.js
- Framework Vue
- Node-sass

### Pré-requis:

Lancez la commande suivante:
`npm install`

Une fois l'installation terminée, lancez le serveur avec:
`npm run dev`

Par défaut, le serveur est accessible en local via le port 8080
